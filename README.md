# Piggy Lib

A shared library providing core infrastructure for all Piggy mods. It simplifies mod development by centralizing common features:

*   **Networking**: Shared payload systems for configuration synchronization and multiplayer state.
*   **UI Components**: Unified radial menus, config screen factories, and feedback overlays.
*   **Utilities**: Common math, block manipulation, and registration helpers.
*   **Anti-Cheat API**: Shared interface for server-side feature enforcement and client feedback.

This library is required by Piggy Admin, Piggy Build, and Piggy Inventory.
